# Aplikacja-Wsparcie-dla-nastroju-
„Wsparcie dla nastroju” to aplikacja do codziennego monitorowania emocji i nastroju. Umożliwia rejestrowanie uczuć, korzystanie z technik relaksacyjnych, motywacyjnych cytatów oraz szybki dostęp do wsparcia w kryzysie. Pomaga użytkownikom lepiej rozumieć siebie, redukować stres i dbać o zdrowie psychiczne w praktyczny sposób.
